### Individual Sponsors

Individual sponsors help sustain the project and show appreciation for the work. Every contribution, no matter the size, makes a difference.

**Ways to contribute:**

- One-time donations
- Contributing code and documentation

Organizations can sponsor UrbanReflex to support open-source smart city development and gain visibility in the community.

**Benefits for organizational sponsors:**
- Logo placement on project website
- Recognition in release notes
- Priority support for issues
- Custom feature development (contact us)

**Contact us for organizational sponsorship:**
- Email: contact@urbanreflex.dev
