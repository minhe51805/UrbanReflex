# Data Models

## NGSI-LD Entities

### RoadSegment

```json
{
  "id": "urn:ngsi-ld:RoadSegment:001",
  "type": "RoadSegment",
  "name": {"type": "Property", "value": "Nguyen Hue Blvd"},
  "location": {
    "type": "GeoProperty",
    "value": {"type": "LineString", "coordinates": [[106.69, 10.77]]}
  },
  "trafficFlow": {"type": "Property", "value": "heavy"}
}
```

### Streetlight

```json
{
  "id": "urn:ngsi-ld:Streetlight:001",
  "type": "Streetlight",
  "status": {"type": "Property", "value": "on"},
  "powerConsumption": {"type": "Property", "value": 25.5, "unitCode": "WTT"}
}
```
